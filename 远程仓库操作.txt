创建远程仓库
网址：https://github.com
创建github账户
创建远程仓库

执行SSH钥匙对，将公钥绑定至github上


本地添加远程仓库并建立关联
语法：
git remote add origin（git默认关键字，可更改） 协议名称@github网站：账户/仓库名
git remote add origin git@github.com:liuxc083/gitlearn.git


把本地库的所有内容推送到远程库：
git push -u origin master






创建并切换至新的分支
语法：
git checkout -b（创建并切换至新分支） 分支名称
[root@localhost gitlearn]# git checkout -b ops_dev
Switched to a new branch 'ops_dev'

切换分支
git checkout 分支名称

查看分支
git branch
[root@localhost gitlearn]# git branch
  main
* ops_dev

合并分支
git merge 分支名称
注意，在主分支上合并之前创建的分支

删除分支
git branch -d dev

小结：
-b <new_branch>
    Create a new branch named <new_branch> and start it at <start_point>; 
	表示创建新分支
-d 参数 表示删除
